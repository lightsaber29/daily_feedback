# 📘 감정 + 학습 일지 자동화 시스템

Notion과 OpenAI ChatGPT API를 연동하여 하루 학습 일지를 요약하고, 자동 피드백을 생성하는 루틴 자동화 시스템입니다.

## ✅ 주요 기능
- ✅ 감정 + 신체 상태 + 학습 피드백 템플릿 제공
- ✅ Notion DB → GPT API → Notion 업데이트 자동화
- ✅ 프롬프트 커스터마이징 가능

## 향후 업데이트 목표 항목
- bold, 줄바꿈 처리 추가
- Node.js 코드에 1일 실행 횟수 제한 로직
- 프롬프트 길이 자동 줄이기 (토큰 절약)
- 사용량 로그 기록 스크립트
- 입출력 토큰 수 계산기 추가
- GPT 피드백 품질 튜닝 (말투, 길이 등)
(선택사항)
- 체크박스 트리거
- 버튼처럼 보이는 링크
